# One For All

Suite gratuita de entretenimento e edição de mídia no navegador: **OnePic**, **OneDes**, **OneRed**, **OneTon**, **OneMov** e **OneMus**.

## Modos

| Modo | Função |
|------|--------|
| OnePic | Remoção de fundo, pincel, ferramentas de imagem |
| OneDes | Desenho e edição |
| OneRed | Redimensionamento |
| OneTon | Matiz e saturação em lote |
| OneMov | Manipulação e exportação de vídeo |
| OneMus | Download de áudio do YouTube *(desativado na versão pública — em breve)* |

## Uso

- **Edição (OnePic–OneMov):** abra `index.html` ou publique no GitHub Pages — o processamento ocorre no seu navegador; arquivos ficam no seu dispositivo.
- **OneMus:** requer servidor com API (`onemus-server/`). Veja `render.yaml` para deploy no [Render](https://render.com).
